VM1 DOC

  Endianness: Little endian 
  Bitwidth:   16bit
  Memory:     128kb

OP CODES

  Assembly keyword | Hex value | Parameters

  End program execution:
  END | 00 | none

  Jump to other memory location:
  JMP | 01 | register

  Branch with given flag:
  BRN | 02 | flag, register

  Addition:
  ADD | 03 | register, register

  Subtraction:
  SUB | 04 | register, register
    
  Multiplication:
  MUL | 05 | register, register
  
  Division:
  DIV | 06 | register, register

  Remainder:
  REM | 07 | register, register

  Set register value with value:
  SRV | 08 | register, value
 
  Set register value with another register value:
  SRR | 09 | register, register

  Set register value with value from memory address:
  SRM | 0A | register, memory address

  Set value in memory address with value from register:
  SMR | 0B | memory address, register

  Puts contents of register into console output with given format:
  PUT | 0C | register, register
